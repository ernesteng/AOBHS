This is the release page for the code and scripts accompanying our working paper, Adaptive Online Bagging with Hybrid Sampling for Non-Stationary Imbalanced Data Streams: A Case Study on Credit Card Fraud Detection.

AOBHS is fully compatible with the Scikit-multiflow framework. (https://github.com/scikit-multiflow/scikit-multiflow)

The generator code and the datasets are from the Github repository: https://github.com/alessiobernardo/SMOTE-OB/tree/main (Bernardo & Valle, 2021).

The MOA results are conducted on the MOA platform, see https://github.com/Waikato/moa.
